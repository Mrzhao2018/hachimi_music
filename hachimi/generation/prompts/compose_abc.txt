## ABC Notation Rules
- Start with: X:1, T:{title}, M:{time_signature}, L:1/8, Q:1/4={tempo}, K:{key}
- Declare all voices in the header: V:1 name="Piano" clef=treble, V:2 name="Cello" clef=bass, etc.
- Then use inline [V:n] markers to interleave voices. Example format:
  [V:1] measure1 | measure2 | measure3 | measure4 |
  [V:2] measure1 | measure2 | measure3 | measure4 |
  [V:1] measure5 | measure6 | measure7 | measure8 |
  [V:2] measure5 | measure6 | measure7 | measure8 |
- Standard note names: C D E F G A B c d e f g a b
- Octave: C,=below middle, C=middle, c=above, c'=two above
- Accidentals: ^F (sharp), _B (flat), =F (natural)
- Note lengths: A2 (double), A/2 (half), A3/2 (dotted)
- Rests: z, z2, z4
- Bar lines: | (bar), |] (final bar on LAST measure only)
- **Do NOT use repeat signs** (|: :|) — write out all measures explicitly
- Chords: [CEG]

## Output
Output ONLY the raw ABC notation. No JSON, no markdown fences, no explanation. Just the ABC notation text starting with X:1.
